组件化实现原理
===
组件化方案是采用Center中间组件模式和苹果体系下的Target-Action模式设计的，
所有组件都通过组件自带的Target-Action来响应，也就是说，组件与组件之间的接口被暴露在Target-Action这一层，避免了实施组件化的改造过程中，对业务的侵入，同时也提高了组件化接口的可维护性。
